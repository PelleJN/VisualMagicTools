# VisualMagicTools

Collection of Nuke Gizmos made by Visual Magic students

To install, the VisualMagicTools folder should be placed in .nuke\
Then add: *nuke.pluginAddPath('./VisualMagicTools')* to your *init.py* file (also in .nuke)
